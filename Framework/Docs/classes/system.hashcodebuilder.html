Class HashCodeBuilder

Class providing means to implement the Object::hashCode method in a concise and consistent way. This class is consistent with the HashCodeBuilder class, so that equal objects are guaranteed to return the same hash code value.

appendBase

  • appendBase(baseHashCode: number): HashCodeBuilder
    • Defined in System/HashCodeBuilder.ts:78

    Adds the result of super.hashCode() to this builder.

    Always call appendBase if the class that uses this HashCodeBuilder for the implementation of its Object::hashCode method inherits from some superclass. This ensures that all properties of the superclass are conviniently incorporated into the final result of the hashCode method. Accordingly, only the own properties (used for the equals and hashCode methods) of the class must be appended. This means that each class may concentrate on their own properties.

    Parameters

    • baseHashCode: number

      The result when calling {super.hashCode()}.

    Returns HashCodeBuilder

    This HashCodeBuilder instance, which may be used for chaining.
