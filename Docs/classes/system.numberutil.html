Static equalsNDigitsPrecision

  • equalsNDigitsPrecision(first: number, second: number, precisionDigits?: number): boolean
    • Defined in System/NumberUtil.ts:60

    Gets whether the first number equals the second number. The two values are being considered equal if the first n digits of their difference are all zero.

    throws

    {ArgumentOutOfRangeException} If the {precisionDigits} argument is negative.

    Parameters

    • first: number
    • second: number
    • Default value precisionDigits: number = 10

      The number of decimal digits that must be zero in the absolute difference |{first} - {second}| in order to be considered equal. If {precisionDigits} is zero, the two values must match exactly to be treated equal. Must be non-negative. Defaults to 10.

    Returns boolean

    true, if the the two numbers are equal (with respect to the given {precisionDigits} argument), false otherwise.
