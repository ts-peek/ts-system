Class EqualsBuilder

Class providing means to implement the Object::equals method in a concise and consistent way. This class is consistend with the HashCodeBuilder class, so that equal objects are guaranteed to return the same hash code value.

appendBase

  • appendBase(baseEquals: boolean): EqualsBuilder
    • Defined in System/EqualsBuilder.ts:47

    Adds the result of super.equals(object) to this builder.

    Always call appendBase if the class that uses this EqualsBuilder for the implementation of its Object::equals method inherits from some superclass. This ensures that all properties of the superclass are conviniently incorporated into the final result of the equals method. Accordingly, only the own properties (used for the equals and hashCode methods) of the class must be appended. This means that each class may concentrate on their own properties.

    Parameters

    • baseEquals: boolean

      The result when calling super.equals(object).

    Returns EqualsBuilder

    This EqualsBuilder instance, which may be used for chaining.

    Tests if two numbers are equal, optionally using a the number of precision digits. This method is compatible with the hash code generated by HashCodeBuilder when being invoked with the same value for precisionDigits.

    throws

    ArgumentOutOfRangeException If the precisionDigits argument is negative.

    Parameters

    • first: number

      The first number to compare.

    • second: number

      The second number to compare.

    • Default value precisionDigits: number = 0

      The number of decimal digits that must be zero in the absolute difference |first - second| in order to be considered as equal. If precisionDigits is zero, the two numbers must match exactly to be treated as equal. Must be non-negative. Defaults to 0.
